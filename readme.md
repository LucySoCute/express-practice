#express 練習
---
此專案使用 express 搭配 express-handlebars

## Features
---
點擊導覽列切換路由，顯示不同頁面內容

## Environment
---
* express: 4.18.2
* express-handlebars: 6.0.6
* nodemon: 2.0.20

## Install process
1. 打開終端機，下載此專案。
2. 進入專案資料夾
3. 安裝套件
4. 執行 nodemon
5. 終端機顯示 Server is running on localhost:3000 代表連結成功
6. 打開瀏覽器並輸入 localhost:3000
